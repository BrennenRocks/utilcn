# repeatPad

Pads a string evenly on both sides until it reaches the desired length, creating a centered effect.

## Usage

```typescript
import { repeatPad } from '@/registry/default/strings/repeat-pad';

const result = repeatPad("TS", 6, "-");
console.log(result); // "--TS--"
```

## Parameters

| Parameter | Type     | Default | Description                                    |
|-----------|----------|---------|------------------------------------------------|
| `str`     | `string` | -       | The input string                               |
| `length`  | `number` | -       | Desired total length                           |
| `pad`     | `string` | `' '`   | The character/string used as padding           |

## Returns

`string` - The padded string centered with the given padding.

## Examples

```typescript
repeatPad("TS", 6, "-"); // "--TS--"
repeatPad("Hello", 11, " "); // "   Hello   "
repeatPad("JS", 5, "*"); // "*JS**" (odd padding goes to the right)
repeatPad("Test", 10, "="); // "===Test==="
repeatPad("A", 7, "0"); // "000A000"
```

## Implementation

```typescript
export function repeatPad(str: string, length: number, pad = ' '): string {
  const total = Math.max(length - str.length, 0);
  const left = Math.floor(total / 2);
  const right = total - left;
  return pad.repeat(left) + str + pad.repeat(right);
}
```

## How It Works

1. **Calculate Total Padding**: Determines how much padding is needed
2. **Split Padding**: Divides the padding between left and right sides
3. **Handle Odd Padding**: When padding is odd, the extra character goes to the right side
4. **Apply Padding**: Uses `String.repeat()` to add padding on both sides

## Edge Cases

- If the desired length is less than or equal to the string length, returns the original string
- When total padding is odd, the extra padding character is added to the right side
- Works with multi-character padding strings
- Returns the original string if the target length is smaller than the input string

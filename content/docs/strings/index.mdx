# String Utilities

A comprehensive collection of utility functions for string manipulation and transformation. These utilities handle common string operations with proper edge case handling and TypeScript support.

## Features

- **TypeScript Support**: All functions are fully typed
- **Edge Case Handling**: Proper handling of empty strings, special characters, and edge cases
- **Acronym Support**: Case conversion functions properly handle acronyms (e.g., "XMLHttpRequest")
- **Performance Optimized**: Efficient implementations using native JavaScript methods
- **Zero Dependencies**: No external dependencies required

## Installation

These utilities are part of the utilcn registry. You can copy the individual functions you need into your project.

## Usage Pattern

All string utilities follow a consistent pattern:

```typescript
import { functionName } from '@/registry/default/strings/function-name';

const result = functionName(inputString, ...additionalParams);
```

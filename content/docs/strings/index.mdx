## Available Functions

### Case Conversion

<Cards>
  <Card title="capitalize" href="/docs/strings/capitalize">
    Capitalizes the first character of a string
  </Card>
  <Card title="capitalizeAll" href="/docs/strings/capitalize-all">
    Capitalizes the first letter of every word (title case)
  </Card>
  <Card title="toCamelCase" href="/docs/strings/to-camel-case">
    Converts strings to camelCase with acronym handling
  </Card>
  <Card title="toPascalCase" href="/docs/strings/to-pascal-case">
    Converts strings to PascalCase with acronym handling
  </Card>
  <Card title="toKebabCase" href="/docs/strings/to-kebab-case">
    Converts strings to kebab-case
  </Card>
  <Card title="toSnakeCase" href="/docs/strings/to-snake-case">
    Converts strings to snake_case
  </Card>
</Cards>

### String Manipulation

<Cards>
  <Card title="repeatPad" href="/docs/strings/repeat-pad">
    Pads a string evenly on both sides for centering
  </Card>
  <Card title="truncate" href="/docs/strings/truncate-string">
    Truncates strings with ellipsis when they exceed length
  </Card>
  <Card title="reverseString" href="/docs/strings/reverse-string">
    Reverses the characters in a string
  </Card>
</Cards>

## Installation

These utilities are part of the utilcn registry. You can copy the individual functions you need into your project.
